ec800x_test.py
这是一个基于 EC800X 4G通信模块 的物联网（IoT）演示程序，主要用于传感器数据采集、传输与远程监控.
  1. 串口通信控制
通过串口（如COM3）连接EC800X模组
发送AT指令控制模组行为（网络连接、数据传输等）
  2. 网络连接与配置
自动执行一系列AT命令初始化4G网络
检查SIM卡状态、信号强度、网络附着状态
激活PDP上下文，建立TCP连接
  3. 模拟传感器数据生成
生成模拟的环境传感器数据（温度、湿度、气压、电压等）
支持自定义设备ID、位置、时间戳等信息
  4. 数据上传至服务器
将传感器数据以JSON格式通过HTTP POST发送到远程服务器
演示使用 httpbin.org/post 作为测试端点
支持自定义服务器地址和API路径（注释中已预留）
  5. 本地数据存储
使用 database_manager.py将数据保存到本地SQLite数据库
支持查询统计数据（如记录数、平均温湿度）
  6. 物联网场景模拟
模拟多个监测周期（如5个周期）
每个周期生成数据并上传，模拟服务器响应
支持服务器下发指令（如调整采样频率）
  7. 可视化支持（注释中）
预留 data_visualizer.py 接口用于数据可视化
可显示实时仪表盘、温湿度对比图、数据统计等

database_manager.py
这是 传感器数据管理系统 的数据库模块，专门用于存储、管理和查询EC800X物联网项目中的传感器数据。
  1. 数据库初始化
自动创建SQLite数据库文件（sensor_data.db）
创建三张核心表：
devices - 设备信息表:
                    记录每个设备的基本信息和状态
                    自动更新最后在线时间
sensor_data - 传感器数据表（主表）:
                    存储所有传感器读数
                    包含时间戳和设备关联
device_commands - 设备命令表（用于双向控制）:
                    实现服务器到设备的双向通信
                    支持命令排队和执行状态跟踪
创建索引优化查询性能
  3. 数据存储管理
保存传感器数据：将JSON格式的传感器数据存入数据库
设备信息同步：自动更新设备最后在线时间和状态
原始值存储：将raw_values字典存储为JSON字符串
  4. 数据查询与检索
获取最新数据：按设备ID查询最近的N条记录
时间范围查询：支持按时间段筛选数据
数据统计：计算平均值、总数、时间范围等
  5. 设备命令管理
命令队列：存储服务器下发给设备的指令
状态跟踪：命令状态（pending/executed）
命令执行记录：记录执行时间
  6. 数据可视化与导出
仪表盘显示：在控制台显示设备状态和数据概览
CSV导出：将数据导出为CSV文件用于进一步分析
